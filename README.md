🌍 EarthMirror AI

Track, Understand & Reduce Your Carbon Footprint

EarthMirror AI is an AI-powered sustainability platform designed to help individuals understand their environmental impact, track their carbon footprint, and receive personalized recommendations for a greener future.

Built using Flask, Gemini AI, HTML, CSS, and JavaScript.

---

🚀 Features

Carbon Footprint Calculator

Estimate your carbon footprint based on:

- Travel habits
- Electricity usage
- Food consumption
- Shopping habits

AI Sustainability Coach

Powered by Gemini AI.

Provides:

- Carbon footprint analysis
- Personalized sustainability recommendations
- Weekly eco challenges
- Actionable lifestyle improvements

Future Earth Simulator

Visualize:

- Future environmental impact if current habits continue
- Potential improvements through sustainable practices

Responsive Design

- Mobile-friendly
- Modern UI
- Clean user experience

⚙️ Installation

Clone Repository

git clone https://github.com/yourusername/earthmirror-ai.git

cd earthmirror-ai

Create Virtual Environment

python -m venv venv

Activate Virtual Environment

Windows:

venv\Scripts\activate

Mac/Linux:

source venv/bin/activate

Install Dependencies

pip install -r requirements.txt

---

🔑 Environment Variables

Create a .env file:

GEMINI_API_KEY=YOUR_API_KEY_HERE

---

▶️ Run Locally

python app.py

Application will run on:

http://127.0.0.1:5000
